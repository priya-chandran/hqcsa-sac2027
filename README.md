HQCSA GitHub Pages Ready Website
=================================

This folder contains the complete HQCSA website as a single self-contained HTML file.

Main file:
- index.html

How to preview:
1. Open index.html in any browser.

How to publish on GitHub Pages:
1. Create a GitHub repository, for example:
   hqcsa-sac2027
2. Upload index.html to the repository root.
3. In GitHub, go to:
   Settings → Pages
4. Under "Build and deployment":
   Source: Deploy from a branch
   Branch: main
   Folder: /root
5. Click Save.
6. GitHub will generate a public URL, usually like:
   https://<username>.github.io/hqcsa-sac2027/

What is included:
- Full homepage
- CFP
- Technical areas
- SAC 2027 dates
- Submission information
- Official SAC 2027 link
- Policy notes
- Track chairs with email and faculty profile links
- TPC placeholder
- News
- Contact

- Official paper submission link when activated
- Confirmed TPC members
- Any updated SAC 2027 dates

Google Sites usage:
Use Google Sites as a simple landing page if required. Add a button:
"Visit HQCSA Website"
and link it to the GitHub Pages URL.
